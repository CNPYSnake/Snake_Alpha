# 中蟒Snake编译器设计文档

## 一、项目概述

中蟒Snake是一个支持中文语法的Python编译器，基于Lark Parser实现，提供完整的中文编程体验。

**版本**: Alpha1.2.6

### 1.1 设计目标

- **语言规范**：定义完整的语法规范，支持变量、控制流、函数、类、类型注解等核心特性
- **多语言支持**：支持7种语言关键词（中文、英文、日文、韩文、法文、德文、西班牙文），可通过配置文件切换
- **简易IDE**：提供基于PyQt6的简易IDLE，支持代码编辑、运行和语法高亮
- **标准库映射**：映射28个常用Python标准库到中文模块名
- **语义分析**：提供未定义变量/函数检测等语义检查功能

### 1.2 技术栈

| 组件    | 技术          | 版本     |
| ----- | ----------- | ------ |
| 词法分析  | Lark Parser | 0.12.0 |
| GUI框架 | PyQt6       | 6.x    |
| 配置管理  | JSON        | -      |

## 三、实现架构

### 3.1 编译流程

```
源代码(.sn) → 词法分析(Lark) → 语法分析(LALR) → AST → 语义分析 → 代码生成(Transformer) → Python源码 → exec()执行
```

### 3.2 模块结构

```
中蟒snake/
├── core.py           # 核心模块（配置、翻译、解析器、代码生成器、语义分析器）
├── idle.py           # PyQt6 IDE界面
├── lang_config.json  # 多语言配置文件（7种语言）
├── grammar.lark      # Lark语法文件
└── docs/
    ├── 中蟒设计文档.md
    ├── 开发文档.md
    └── 设计文稿
```

### 3.3 核心组件

#### 3.3.1 配置模块

处理路径配置，支持开发环境和打包环境：

```python
def get_base_dir():
    """获取程序基础目录"""
    ...

def get_save_dir():
    """获取保存目录（C:\snake）"""
    ...
```

#### 3.3.2 翻译模块

处理多语言翻译，包括错误信息、UI文本和关键词映射：

```python
class Translator:
    def translate_error(error_message):
        """翻译错误信息为中文"""
        ...
    
    def get_ui_text(key):
        """获取UI文本"""
        ...
```

#### 3.3.3 词法分析器

使用Lark的lexer进行词法分析，支持动态关键词加载：

```python
from lark import Lark

grammar = f"""
    if_stmt: "{kw['if']}" expression ":" block
    ...
"""
parser = Lark(grammar, parser='lalr', lexer='standard')
```

#### 3.3.4 语法分析器

使用Lark的LALR(1)解析器构建AST，支持中文关键词和运算符。

#### 3.3.5 代码生成器

使用Transformer模式将AST转换为Python源码：

```python
class PythonCodeGenerator(Transformer):
    def if_stmt(self, items):
        return f"if {items[0]}:\n    {items[1]}"
```

#### 3.3.6 语义分析器

提供符号收集和引用验证功能：

```python
class SemanticAnalyzer:
    def analyze(self, tree):
        """分析AST，返回错误列表"""
        ...
```

```python
def build_grammar(self):

        ...
    """
```
